## 📌 Sobre

Este projeto apresenta 10 comparações entre código de má qualidade (`ruim.java`) e uma versão melhor (`bom.java`), aplicando:

- Clean Code
- SOLID quando faz sentido
- Separação de responsabilidades
- Injeção de dependência
- Tratamento de erros
- Segurança e eficiência

Cada pasta contém:

- `ruim.java`
- `bom.java`
- `README.md` explicativo

## ▶ Como executar

Exemplo (Windows PowerShell):

```powershell
Set-Location .\01-legibilidade
javac .\bom.java
java ExemploLegibilidadeBom
```

Exemplo (Prompt de Comando - CMD):

```cmd
cd 01-legibilidade
javac bom.java
java ExemploLegibilidadeBom
```

Exemplo (Linux/macOS - Bash):

```bash
cd 01-legibilidade
javac bom.java
java ExemploLegibilidadeBom
```

Você também pode compilar `ruim.java` para comparar os estilos.
